# Projeto SQL – Análise de Vendas de Supermercado

## Objetivo
Analisar dados de vendas de um supermercado utilizando SQL para identificar produtos, categorias e cidades com maior faturamento.

## Ferramentas
- MySQL
- SQL

## Análises realizadas
- Faturamento total do supermercado
- Categorias mais lucrativas
- Produtos com maior faturamento
- Cidades com maior volume de vendas
- Cliente mais ativo
- Ticket médio

## Conclusão
A análise permite entender o comportamento de compra dos clientes e identificar categorias estratégicas para aumento de receita.
